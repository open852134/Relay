### 產品介紹

Relay 驛站停車場  
透過網站建置，讓外出旅人可以清楚知道目的地是否有停車場
也藉此讓業主可以招商，讓有意願的地主也可加盟停車場行列  


本專案使用單純CSS3作為開發  
並沒有使用任何框架輔助(完全手刻客製化)  
設計上使用單頁輪播的感覺作為畫面切換搭配  
GoogleMap API 讓使用著知道各個停車場所在位置


### 開發技術

* CSS3
* Javascrpt
* Jquery
